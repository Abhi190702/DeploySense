# DeploySense v0.1.0 Release Notes

DeploySense v0.1.0 is the first public beta release of the DevOps intelligence platform.

It includes:

- Dockerfile, GitHub Actions, Kubernetes, and Docker Compose scanners
- Log Doctor deployment error explanations
- CLI, API, web dashboard, GitHub Action, and VS Code MVP
- SARIF output, fail-on thresholds, project scans, and safe auto-fixes
- npm-publish-ready package checks for all runtime packages

This release is intended for early adopters and contributors while the project gathers real-world feedback before v1.0.0.
