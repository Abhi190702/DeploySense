# K8S_NO_READINESS_PROBE

Deployment missing readiness probe

- Severity: high
- Category: reliability
- Auto-fixable: no
- Tags: probes

## Description

This rule detects a deployment risk in reliability and reports a plain-English fix through the DeploySense scanner.

## How to Fix
