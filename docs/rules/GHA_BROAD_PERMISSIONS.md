# GHA_BROAD_PERMISSIONS

Workflow permissions are too broad or implicit

- Severity: high
- Category: security
- Auto-fixable: no
- Tags: permissions

## Description

This rule detects a deployment risk in security and reports a plain-English fix through the DeploySense scanner.

## How to Fix
