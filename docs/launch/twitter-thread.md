1/ I launched DeploySense: open-source DevOps intelligence for deployment configs.

2/ It scans Dockerfiles, Kubernetes YAML, GitHub Actions, Compose, and logs.

3/ You get scores, risk categories, plain-English explanations, and safe fixes.

4/ CLI: npx deploysense scan .

5/ It also outputs SARIF for code scanning.

6/ There is a Next.js dashboard for live scans.

7/ The GitHub Action comments on PRs.

8/ Log Doctor explains common deployment failures.

9/ Contributors can add rules quickly.

10/ GitHub: https://github.com/Abhi190702/DeploySense
