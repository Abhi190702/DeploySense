I just launched DeploySense, an open-source DevOps doctor.

It scans Dockerfiles, Kubernetes YAML, GitHub Actions workflows, Docker Compose files, and deployment logs.

It gives health scores, risk categories, plain-English explanations, safe fixes, and GitHub PR comment reports.

Built with TypeScript, Next.js, Express, and 46+ scanner rules.

GitHub: https://github.com/Abhi190702/DeploySense
CLI: npx deploysense scan .
